# Playbook — cong3 STUDIO

Universal work method for all cong3 STUDIO projects.

## What this is

This repo defines **how we work** — principles, validation gates, session protocol, contribution model, and conventions. It is project-agnostic. Each project has its own `project-context.md` for specifics.

## Usage

Injected as a git submodule at `_playbook/` in any project repo. Agents load `_playbook/CLAUDE.md` at session start.

## Files

- `CLAUDE.md` — the operational document (loaded by agents)
- `README.md` — this file (for humans on GitHub)
